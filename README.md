# Pipeline ETL com Python – Santander Bootcamp

Projeto desenvolvido como parte do Santander Bootcamp, com foco na construção
de um pipeline de ETL (Extração, Transformação e Carregamento) utilizando Python.

## Descrição
O pipeline lê um arquivo CSV contendo dados de clientes, gera mensagens
personalizadas como enriquecimento dos dados e salva o resultado em um novo
arquivo CSV.

## Tecnologias utilizadas
- Python
- Pandas

## Como executar
```bash
pip install pandas
python ETL.py
